# AL

A simple Flappy Bird-style browser game built with HTML, CSS, and JavaScript.

## How to play

1. Open `index.html` in a web browser.
2. Press `Space` or click/tap the canvas to make the bird flap.
3. Avoid the pipes and try to score as many points as possible.

## Files

- `index.html` — game structure and canvas.
- `style.css` — visual styling and layout.
- `script.js` — game logic, physics, and rendering.
